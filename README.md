## Featured projects

### 1. LLM Fine‑Tuning Pipeline  
A production‑ready pipeline for training and evaluating large language models.

- Full and parameter‑efficient fine‑tuning (PEFT / LoRA)
- Data preparation, evaluation scripts, and inference utilities
- Streamlit demo app, Dockerfile, and structured configs for real deployments

Repo: [`llm-finetuning-pipeline`](https://github.com/Darknivht/llm-finetuning-pipeline)

---

### 2. AI‑Powered PDF Research Assistant  

A RAG application for deep research across multiple PDFs.

- Upload multiple PDFs and ask questions in natural language
- Intelligent chunking, FAISS vector search, and precise source citations
- Supports OpenAI, OpenRouter (including free models), and local models
- Streamlit UI and Dockerized deployment for easy hosting

Repo: [`pdf-research-assistant`](https://github.com/Darknivht/pdf-research-assistant)

---

### 3. RAG Chatbot  

A document‑aware chatbot built for both experimentation and production.

- RAG architecture with embeddings, retriever, and generator modules
- Streamlit front end with conversation memory and clean UI
- Supports OpenAI and OpenRouter (free and paid models)
- Docker and Docker Compose setups for simple deployment

Repo: [`rag-chatbot`](https://github.com/Darknivht/rag-chatbot)

---

### 4. Vintage E‑commerce Platform  

A full e‑commerce project with a rich back end and modern front end.

- Python‑first stack with clear separation of apps (store, customer, vendor, auth)
- Templates, static assets, and diagrams for the overall architecture
- Setup guides, upgrade summaries, and test files for key flows

Repo: [`Vintage-Ecommerce`](https://github.com/Darknivht/Vintage-Ecommerce)

---
